# Quantum Machine Learning with Qiskit

This repository contains my hands-on implementations and experiments from the
"Quantum Machine Learning" course provided by the
Hasso Plattner Institute (HPI).

The notebooks explore the fundamentals of Quantum Machine Learning (QML)
through practical coding exercises using Qiskit, hybrid quantum-classical
models, and variational quantum circuits.

---

## Course Information

- **Course:** Quantum Machine Learning
- **Platform:** HPI Open Courses
- **Framework:** Qiskit
- **Topics Covered:**
  - Quantum Support Vector Machines
  - Variational Quantum Circuits
  - Quantum Neural Networks (QNNs)
  - Hybrid Quantum-Classical Learning
  - Feature Maps and Quantum Kernels
  - Optimization in Quantum ML

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Quantum-Machine-Learning-with-Qiskit.git
cd Quantum-Machine-Learning-with-Qiskit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Or manually install:

```bash
pip install qiskit
pip install qiskit-machine-learning
pip install matplotlib
pip install scikit-learn
pip install torch
pip install pylatexenc
```
